# Wine Classification using Naive Bayes

This repository contains a notebook that applies Naive Bayes classifiers to the Wine dataset.

## File
- `wine-classification-naive-bayes.ipynb`

## What this notebook does
- Loads and explores the Wine dataset
- Splits data into training and testing sets
- Trains Gaussian Naive Bayes and Multinomial Naive Bayes models
- Compares model performance
- Summarizes findings in a conclusion section

## How to run
1. Open the notebook in Jupyter Notebook or VS Code.
2. Run the cells from top to bottom.

## Requirements
- Python 3.x
- scikit-learn
- pandas
- matplotlib
- seaborn
- numpy
